## 项目特性
### 📱语音通话模式
  可根据需要选择对讲机或电话模式。
### ⏰ 挂断时间
  可自行设定挂断的时间，节约资源。
### 🎶 音色与人格指令
  支持多种音色，支持自定义人格指令，适配对话场景。
### 🌓 暗色模式
  支持暗色模式，保护您的眼睛。
### 🌍 多语言支持
  - 中文界面
  - English Interface
  - 日本語インターフェース

## 开发&部署
1. 克隆项目 `git clone https://github.com/302ai/302_voice_call`
2. 安装依赖 `pnpm install`
3. 配置302的API KEY 参考.env.example
4. 运行项目 `pnpm dev`
5. 打包部署 `docker build -t realtime . && docker run -p 3000:3000 realtime`
